\subsection{{\href{https://tessera-rag-ai.vercel.app}{Tessera} \hfill 2025}}
\begin{zitemize}
  \item Production RAG platform where users upload documents and receive \textbf{citation-backed answers with zero hallucination} via constraint-based LangChain agent design forbidden from answering without invoking the \texttt{kb\_search} tool.
  \item Engineered \textbf{semantic embedding cache} via CacheBackedEmbeddings to MongoDB KV store: re-upload time drops from \textbf{2.31s to 421ms} (82\% reduction) with namespace-based deduplication.
  \item \textbf{Lighthouse scores}: Performance 92, Accessibility 96, Best Practices 100, SEO 100.
  \item \textbf{Tech}: Next.js 16, Express, MongoDB Atlas, LangChain, Vector Search, Zod, Framer Motion, Spline
\end{zitemize}

\subsection{{\href{https://axiom-ai-justuzair.vercel.app}{AXIOM} \hfill 2025}}
\begin{zitemize}

  \item \textbf{Tech}: TypeScript, Next.js, Express, LangChain LCEL, LightRAG, Tavily, Vercel Serverless Functions
\end{zitemize}
